# Golang Tutorial Project

This is a Golang tutorial project for you to learn Golang in a comprehensive way.

## Overview

Welcome to my project! This README file provides you the comprehensive details about the project, its purpose, functionality, setup instructions, and usage guidelines for learning Golang from the beginning to advance.

## Description

My project is designed for people who want to learn Golang with or without any experiences. It incorporates Golang version 1.25.6, Visual Studio Code, and Git.

## Prerequisites

Before you begin, ensure you have met the following requirements:

1. Go installer installed on your system whichever version you can use. In this project, the contributor uses version 1.25.6. [Click here to download](https://go.dev/doc/install).
2.  Visual Studio Code for streamlining code editor with support for development operations like debugging, task running, and version control. [Click here to download](https://code.visualstudio.com/download).
3. Git for managing source code and repository. [Click here to download](https://git-scm.com/downloads).

## Installation

Follow these steps to set up the project on your local machine:

Open your local directory (e.g., \Documents) and clone the repository:

```bash
git clone <repository_url>
```

Navigate to the project directory:

```bash
cd <project_directory>
```

Build the source code:

```bash
go build <file_name>.go
```

Or just run the source code:

```bash
go run <file_name>.go
```
